# 超级培训图表化预览计划

本文只记录当前要做的事实计划。

## 目标

- [x] 排班总览默认先看图和统计，不先看 Excel 明细表。
- [x] Excel 明细表保留，但默认折叠放在最下面。
- [x] 页面同时保留项目视角和人员视角。
- [x] 人员风险清单放在倒数第二个，用于防漏排复核。
- [x] UI 只展示逻辑层输出，不在 UI 中判断业务规则。

## 默认展示顺序

- [x] 状态统计图。
- [x] 项目统计图。
- [x] 项目 × 状态统计表。
- [x] 项目分组清单。
- [x] 人员风险清单。
- [x] Excel 明细表，默认折叠。

## 逻辑输出

- [x] 排班评估核心输出项目状态统计表数据。
- [x] 排班评估核心输出项目分组清单数据。
- [x] 排班评估核心输出人员风险清单数据。
- [x] 人员风险只包含：
  - [x] 已过期
  - [x] 必须排
  - [x] 推荐排
  - [x] 异常

## 文件拆分

- [x] `app-renderers.ts` 不继续承载所有展示逻辑。
- [x] 新增图表渲染模块。
- [x] 新增统计和分组视图渲染模块。
- [x] 新增明细表渲染模块。
- [x] `app-renderers.ts` 只做统一出口。

## 验证

- [x] 更新或补充测试，保护统计表、项目分组、人员风险数据。
- [x] 运行 `npm.cmd run typecheck`。
- [x] 运行 `npm.cmd test`。
