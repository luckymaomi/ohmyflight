# 姓名匹配员工号用户手册

## 解决什么问题

姓名匹配员工号把通知、聊天记录或其他文本中的花名册姓名识别出来，快速得到员工号、分部、技术信息和技术等级，便于复制或导出名单。

## 怎样操作

1. 打开工具，等待默认花名册加载；如加载失败或需要使用新名单，点击重新选择文件。
2. 手动花名册第一行必须包含 `员工号`、`姓名`、`分部`，可选 `技术信息`。
3. 粘贴待识别文本，点击查询匹配。
4. 查看结果中的姓名、员工号、分部和技术信息，重点核对同名人员。
5. 只需要部分人员时先勾选对应行；然后复制某一列或导出 Excel。
6. 没有勾选任何行时，复制和导出会处理全部结果。

## 操作注意事项

- 工具按花名册完整姓名是否出现在文本中匹配，不做模糊搜索或别名识别。
- 同名人员可能同时进入结果，必须结合员工号、分部人工确认。
- 技术等级从技术信息文本中提取，无法识别时留空，不代表没有技术等级。
- 花名册缺少必需表头会停止读取，不会猜固定列。
- 文件只在浏览器本地解析。
