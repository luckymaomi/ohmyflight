# 自动点 OA 助手用户手册

## 解决什么问题

自动点 OA 助手循环打开待办并点击可以直接确认的“已阅”，减少大量常规通知逐条打开的操作。需要填写意见、审批或特殊处理的待办仍由用户完成。

## 使用前准备

- 安装 Python、Playwright、colorama，并运行 `playwright install chromium`。
- 从工具页下载 `app.py`。
- 确认 OA 账号和网络可正常登录。

## 怎样操作

1. 运行 `python app.py`。
2. 选择有限模式或无限模式；有限模式输入计划点击次数。
3. 在打开的浏览器中人工登录 OA。
4. 开始后观察脚本打开待办并点击“已阅”。
5. 遇到督办等无法自动完成的页面时，在浏览器中手动处理，再等待页面刷新或返回列表，让脚本继续。
6. 有限模式达到次数后检查终端统计；无限模式完成本轮工作时由用户中断。

## 操作注意事项

- 自动点击会改变 OA 待办状态，开始前确认当前列表适合批量已阅。
- 不要让脚本替代需要阅读、审批、签署或填写意见的业务判断。
- 脚本运行时避免手动切换它正在使用的页面，除非终端明确要求人工处理。
- 页面升级、登录失效或按钮没有成功响应时，应暂停并检查，不要只看循环次数。
- 工具只输出终端日志，不生成业务结果文件。
