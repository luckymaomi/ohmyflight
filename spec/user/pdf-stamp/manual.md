# PDF 加水印用户手册

## 解决什么问题

PDF 加水印把一张图片按统一或分段规则叠加到 PDF 页面，可分别控制哪些页使用、放在哪里、显示多大和透明程度，适合给手册、通告或内部资料批量加标识。

## 怎样操作

1. 上传一份 PDF 和一张水印图片。
2. 在默认规则中选择全部页、奇数页、偶数页或指定页。
3. 设置页码范围、位置、宽高和透明度。需要保持图片不变形时打开比例锁定。
4. 在页面预览中拖动水印或使用缩放手柄微调，也可直接输入毫米数值。
5. 不同页需要不同样式时点击添加规则，逐条设置。
6. 翻页抽查每种规则的首尾页面。
7. 打开“导出预览”，确认同一页面上所有规则叠加后的最终效果。
8. 点击导出 PDF，下载新文件。

## 操作注意事项

- 指定页支持单页和范围组合，页码不能超过 PDF 总页数。
- 普通编辑预览侧重当前规则；最终效果以“导出预览”为准。
- 多条规则可以同时命中一页，会全部叠加。
- 水印拖动和缩放会限制在页面边界内；仍需检查不同页面尺寸的 PDF。
- 图片透明区域、原始清晰度和宽高比会直接影响导出效果。
- 文件全部在浏览器本地处理，源 PDF 不会被覆盖。
