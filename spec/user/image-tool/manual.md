# 图片工具用户手册

## 解决什么问题

图片工具在浏览器本地完成格式转换、体积压缩、尺寸调整、自由裁剪和 Base64 互转，适合整理上传材料、制作网页图片或处理接口中的图片文本。

## 怎样操作

### 格式转换

上传一张或多张图片，选择目标格式和质量，执行转换后逐张下载。

### 图片压缩

上传图片，设置质量和可选最大宽度。先看页面估算的压缩后大小，再执行并下载带 `_compressed` 的结果。

### 调整尺寸

上传图片，填写目标宽和高。保持比例时图片会完整放进目标范围；关闭保持比例会强制拉伸到指定宽高。结果统一下载为 PNG。

### 图片裁剪

上传一张图片，在编辑区拖动裁剪框，确认后预览裁剪结果并下载 PNG。需要重做时点击重置。

### Base64 互转

图片转 Base64 时上传图片并复制完整 Data URL；Base64 转图片时粘贴完整 Data URL 或裸编码，转换成功后预览并下载 PNG。

## 操作注意事项

- 格式转换、压缩和尺寸调整支持多图；裁剪与 Base64 每次主要处理一张。
- JPEG 不支持透明背景，转换前应确认透明区域的处理效果。
- 保持比例的尺寸调整不会保证宽和高都恰好等于输入值，而是保证图片完整放入范围。
- 裸 Base64 默认按 PNG 解码；无效编码会提示失败。
- 大图和批量处理会占用浏览器内存，处理完成后可清空列表再继续。
- 文件不上传服务器，刷新页面会清空当前处理状态。
