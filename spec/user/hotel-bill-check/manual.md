# 酒店皇帝用户手册

## 解决什么问题

酒店皇帝按“姓名 + 日期容差”把酒店账单与入住登记记录对应起来，帮助发现未登记、重复使用登记记录或日期相差过大的账单，并保留入住证明链接。

## 怎样操作

1. 准备账单表和登记表，先确认各自真正的表头行、姓名列和日期列。
2. 打开工具，上传两份 Excel，并为每份文件选择正确表头行。
3. 在列配置区核对系统识别的字段，分别选择姓名列和日期列。
4. 设置日期容差；默认 `±2天`，应按当前核对业务实际选择。
5. 点击开始匹配，先看已匹配、重复匹配、未匹配数量。
6. 逐条查看登记日期、日期差和证明链接；重复匹配需要确认是否确为多人次或原表重复。
7. 点击导出 Excel 留底。多个入住证明会展开为多个可点击列。

## 操作注意事项

- 表头行或列选错会让结果整体失真，匹配前必须先看预览。
- 同名候选较多时，工具选择容差内日期最近的一条，不判断房间、金额或订单号。
- 同一登记记录被第二条账单再次使用会标记重复匹配，不会自动判定哪条账单错误。
- 日期必须能解析为真实日历日；异常日期应回原表修正。
- 工具不覆盖源文件，文件在浏览器本地处理。
