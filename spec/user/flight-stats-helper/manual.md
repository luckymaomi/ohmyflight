# 飞行经历与左座经历统计用户手册

## 解决什么问题

这个助手按多名员工和各自日期区间批量查询飞行门户中的飞行经历或左座经历与起落数，并汇总到本地 Excel，适合替代重复的逐人查询和抄数。

## 使用前准备

- 安装 Python 3.7+、Playwright、openpyxl、colorama，并运行 `playwright install chromium`。
- 从工具页下载 `app.py` 和查询模板。
- 在模板中准备员工号、姓名、开始日期、结束日期；确认起止日期和查询类型。
- 内网和账号必须能够正常打开飞行门户。

## 怎样操作

1. 在脚本目录运行 `python app.py`。
2. 选择飞行经历或左座经历查询。
3. 选择 Excel 导入或直接粘贴，按提示提供人员和日期区间；如启用白名单，先核对过滤后人数。
4. 在浏览器中人工完成门户登录，并按提示进入查询页面。
5. 开始批量查询，观察终端进度。单条失败时脚本会记录并继续。
6. 完成后打开结果 Excel，核对员工号、姓名、起止时间、经历值和起落总数。
7. 对错误报告中的人员单独回门户复核或整理后重跑。

## 操作注意事项

- 查询前先确认日期边界，飞行经历与左座经历是两个不同口径。
- 自动化依赖当前门户页面，页面改版或加载不完整时可能失败。
- 不要在脚本运行时手工改动它正在查询的页面。
- 空值不一定代表没有经历；如果该行标记错误，应按失败处理而不是按零处理。
- 结果写入新 Excel，不修改查询模板。
