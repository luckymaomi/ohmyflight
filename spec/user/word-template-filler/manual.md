# Word 模板填充器用户手册

## 解决什么问题

Word 模板填充器把一份字段配置和 Word 模板制作成可以独立分发的填写应用。以后使用者不需要改模板本身，只需填写表单就能生成 Word；已有多人 Excel 数据时还可以批量生成多份 Word。

## 制作应用

1. 在页面下载配置模板，按模板说明填写字段名、显示名称、类型、选项、默认值等。
2. 在 `.docx` 中把需要替换的位置写成 `{字段名}`，字段名必须与配置一致。
3. 上传配置 Excel 和 Word 模板。
4. 填写应用名称，查看字段预览，确认字段类型、必填和选项正确。
5. 点击生成应用，下载 ZIP。
6. 解压 ZIP 后先打开随包说明，再打开 HTML 试填一份 Word，确认占位符和格式。

## 使用生成的应用

### 单份填写

1. 打开 ZIP 内 HTML。
2. 按表单填写字段，列表/循环字段按页面增加行。
3. 点击导出，打开生成的 Word 核对结果。

### 批量填写

1. 使用 ZIP 内随包提供的批量 Excel 模板，不要自己另建列。
2. 每行填写一份文档的数据；第一列 `文件标题` 可控制文件名。
3. 在生成应用中导入 Excel，先看行数、预览和错误。
4. 修正无效日期或选项后，批量导出 Word ZIP。

## 操作注意事项

- 配置字段名、Word 占位符和批量模板表头必须一致。
- 含列表/循环字段的应用仍可单份填写，但当前不能批量导入。
- 日期可使用 Excel 日期或完整年月日；`5/11/26` 这类短日期含义不明确，会被拒绝。
- 选项字段必须填写配置中已有选项。
- `文件标题` 为空时使用应用名和序号；标题中不适合文件名的字符应提前清理。
- 生成 ZIP 后应先用样例完整试填，再交给其他人使用。
