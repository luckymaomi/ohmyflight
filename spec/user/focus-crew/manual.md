# 重点人员标注用户手册

## 解决什么问题

重点人员标注把多类重点人员名单叠加到审班表，在命中姓名后追加类别标签并着色，让审班人员能直接看到需要重点关注的人。

## 怎样操作

1. 准备审班表和重点人员表。重点人员表可用不同 sheet 表示不同类别。
2. 上传审班表，选择正确的员工号列和姓名列。
3. 上传重点人员表，逐个确认参与识别的 sheet，并为每个 sheet 选择员工号列和姓名列。
4. 检查页面显示的类别和人员数量，确认没有把说明页当名单页。
5. 点击开始标注，查看命中人数、类别标签和预览。
6. 抽查一人多类别、同名人员和未命中人员。
7. 点击导出 Excel，得到新的标注文件。

## 操作注意事项

- 当前实际标注主要按姓名匹配，员工号不作为最终唯一键；同名人员必须人工复核。
- 一个人命中多个 sheet 时会追加多个类别标签，颜色按既定类别优先级决定。
- 重点人员 sheet 名会成为类别信息，命名应简洁准确。
- SheetJS 导出可能改变日期时间显示格式；页面建议必要时只把标注后的姓名列复制回原审班表。
- 工具不会修改上传原文件。
