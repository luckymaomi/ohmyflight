# PDF 工具用户手册

## 解决什么问题

PDF 工具在本地完成常见文件整理：从 PDF 提取指定页面、按顺序合并多份 PDF、把 PDF 页面转成图片，以及把多张图片按顺序生成 PDF。

## 怎样操作

### 提取页面

1. 选择页面提取，上传一份或多份 PDF。
2. 为每份文件填写要保留的单页或页码范围。
3. 核对总页数和选择页数，执行提取并逐份下载新 PDF。

### 合并 PDF

1. 选择 PDF 合并，上传全部文件。
2. 拖拽文件调整先后顺序。
3. 执行合并，下载一个新 PDF。

### PDF 转图片

1. 上传 PDF，选择图片格式和缩放倍率。
2. 开始转换，等待全部页面渲染。
3. 下载图片或批量 ZIP。

### 图片转 PDF

1. 上传图片并拖拽调整页面顺序。
2. 确认每张图片方向与尺寸。
3. 生成并下载 PDF。

## 操作注意事项

- 页码从 1 开始，填写范围前先核对 PDF 实际总页数。
- 合并和图片转 PDF 都以当前拖拽顺序为准。
- 高倍率、超大 PDF 或很多图片会占用大量浏览器内存，应分批处理。
- 扫描 PDF 可以做页面处理和转图片，但工具不会 OCR 文字。
- 文件只在浏览器本地处理，输出为新文件，不修改源文件。
