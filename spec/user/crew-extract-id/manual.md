# 提取员工号用户手册

## 解决什么问题

提取员工号用于从聊天、通知、名单或其他混杂文本中快速找出所有独立的六位数字，去重排序后形成可以直接粘贴的员工号清单。

## 怎样操作

1. 把原文本粘贴到输入框。
2. 点击“提取六位数字”。
3. 查看结果数量和逐行列表。
4. 点击“一键复制”，粘贴到 Excel、OA 或其他目标位置。
5. 需要重新处理时点击清空输入。

## 操作注意事项

- 工具只判断“独立六位数字”，不连接花名册验证真实性。
- 重复出现的同一号码只保留一次，结果按升序排列。
- 身份证、电话、订单号中的独立六位片段也可能被识别，复制前应人工扫一遍。
- 与更长数字连续相连的六位片段不会命中。
- 所有文本只在当前浏览器页面处理。
