# 培训皇帝用户手册

## 解决什么问题

培训皇帝用于读取“南货航新总表结构”的整本培训总表，集中查看哪些人员已经过期、必须安排、建议安排、虽已排但不能覆盖到期，以及哪些数据需要人工确认。它还可以模拟排班、查看已排培训分布和年度已培训人次、核对 CRM 年度参加情况，并生成有效期更新后的 Excel 副本。

## 使用前准备

- 准备一份完整培训总表，至少包含 `人员信息表`、精确名为 `CRM` 的工作表和各培训项目工作表。
- 确认项目表中的人员、培训日期、备注、`培训信息是否录入` 和 `机器看` 等字段按现行总表口径填写。
- 工具不会覆盖原文件，但正式导出前仍建议保留原始总表备份。

## 怎样操作

1. 打开培训皇帝，导入整本培训总表。
2. 先看 Excel 健康检查。严重问题表示核心结构不可用；警告表示工具可以继续，但数据需要核对；提示用于说明识别到的 sheet 和统计摘要。
3. 在排班总览设置评估开始和结束日期。默认范围是当天到下月底。
4. 先看状态摘要和项目风险矩阵，再点击矩阵数字查看具体人员。可按项目、状态、到期月份和姓名或员工号筛选。
5. 需要试排时，在人员明细勾选人员，填写项目和培训起止日期，加入模拟排班。重新扫描后观察风险是否被覆盖。
6. 在已排培训分布、年度已培训人次和 CRM 区域选择月份或年份，查看各自独立统计。
7. 需要留底时导出当前筛选的排班总览或人员明细。
8. 更新有效期时，选择人员信息表、培训类型和这些项目共同拥有的更新月份，点击一键更新，核对预览后导出新 Excel。

## 怎样理解结果

- `已过期`：当前有效期早于评估开始日，且没有有效补训安排。
- `已过期已排补训`：已经过期，但项目表或模拟排班中已有有效安排。
- `必须排`：已进入到期区间，仍没有能覆盖本轮到期的安排。
- `推荐排`：已进入建议安排窗口。
- `已排未覆盖`：有安排，但培训日期不能覆盖当前这轮到期。
- `正常`：已有安排能够覆盖，默认不显示。
- `异常`：记录状态相互矛盾或关键数据无法可靠判断，需要回原表核对。

## 操作注意事项

- 工具只适用于当前新总表结构，不适合历史分表或预排分表。
- 模拟排班只保存在当前页面，不写回 Excel；导入新文件会清空模拟记录。
- 备注包含 `取消` 的安排不参与覆盖；`机器看 = Y` 只用于有效期更新。
- `航空安保` 与 `TSA` 是两个独立项目，必须分别在对应项目表留记录。
- CRM 只读取精确名为 `CRM` 的 sheet，是年度参加核对，不产生有效期或排班状态。
- 所有导出都基于当前筛选或当前预览，下载前先确认筛选条件。
