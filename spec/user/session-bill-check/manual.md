# 珠海皇帝用户手册

## 解决什么问题

珠海皇帝用于核对场次表和账单表中的姓名人次是否一致，快速找出漏计、多计或只在一侧出现的人员。它只核对姓名人次，不核对金额、日期、课程或单价。

## 怎样操作

1. 准备场次表，确认存在 `学员`、`教员`、`检查员` 表头。
2. 准备账单表，确认具体记录位于精确名为 `全动` 和/或 `理论` 的 sheet，并有 `姓名` 表头。
3. 打开工具，分别上传场次表和账单表。
4. 查看识别摘要，确认读取到的 sheet、表头和两侧总人次符合原表。
5. 开始核对，先看状态统计，再切换“只看差异”或具体差异类型。
6. 点击姓名明细行，核对这个人在场次表和账单表中的来源记录。
7. 确认后导出 Excel，使用明细和来源 sheet 留底或继续处理。

## 操作注意事项

- 同一姓名出现几次就算几人次，不会去重成人数。
- 场次表三类角色会合并；账单只读取 `全动` 和 `理论`，其他汇总 sheet 不参与。
- 英文名只忽略大小写，不做中英文名映射或拼写纠错。
- 姓名里的多余空格、别名或错误标点可能造成差异，应回原表修正后重算。
- 导出固定包含核对汇总、姓名差异、场次拆分和账单姓名四类证据。
