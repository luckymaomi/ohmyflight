# 航线班次统计用户手册

## 解决什么问题

航线班次统计从一个或多个月份排班表中识别花名册姓名，汇总每个人在各条航线出现的班次，并单列完全识别不到的排班内容供复核。

## 怎样操作

1. 准备排班表：每个待统计 sheet 第一行为表头，A 列为航线，其他列为机组文本。
2. 准备机组花名册，姓名应位于第二列；页面也可尝试加载默认花名册。
3. 上传排班表和花名册，确认工作表列表和花名册人数。
4. 勾选需要合并统计的月份 sheet，不参与的说明页或汇总页不要勾选。
5. 点击开始统计，查看人员乘航线矩阵。
6. 打开未匹配内容，按工作表、行号和航线回原排班表核对。
7. 结果确认后导出 Excel。

## 操作注意事项

- 一个机组单元格命中一名花名册人员，就给该人在当前航线加一班；工具不统计飞行小时或航段数。
- 姓名按完整文本包含关系识别，不做错别字和别名纠正。
- 花名册短姓名可能被较长文本包含，关键数据应抽查。
- 只有完全未命中任何姓名的单元格才进入未匹配区；一个单元格部分命中时仍需人工留意原文。
- 多个 sheet 会合并统计，重复勾选同一业务数据会导致重复计数。
