# 重点人员标注开发规格

## 产品边界

`focus-crew` 把多类重点人员名单匹配到审班表，在姓名单元格追加类别标签并设置颜色。它是浏览器本地 Excel 工具，不修改上传原文件。

## 输入契约

- 审班表和重点人员表支持 `.xlsx`、`.xls`。
- 审班表由用户选择员工号列和姓名列。
- 重点人员表可有多个工作表；每个被识别的工作表代表一类人员，用户分别确认员工号列和姓名列。
- 重点人员工作表从第二行读取数据；空姓名不进入匹配集合。

## 标注规则

- 当前实际匹配键为清理首尾空格后的姓名，员工号列用于页面配置和数据查看，不作为主匹配键。
- 同一姓名可以命中多个类别，输出姓名后依次追加所有类别标签。
- 类别颜色按优先顺序决定：重点、一般、预防、三新、长期；命中未知类别时保留标签但不编造颜色规则。
- 工作簿修改发生在内存副本，用户点击导出后才生成新 Excel。

## 输出与限制

- 页面显示命中数量和标注预览。
- 导出文件保留工作表数据并写入姓名标签与单元格样式。
- SheetJS 重新生成工作簿时可能改变原有日期时间显示格式；页面明确提示必要时将导出表姓名列复制回原表。
