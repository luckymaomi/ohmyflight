# 模拟排班

## 功能

模拟排班用于在不修改培训 Excel 的情况下，临时判断一组计划安排能否覆盖当前排班风险。

用户先在项目风险矩阵的人员明细中勾选人员，再填写项目和培训日期，将选中人员批量加入模拟排班。

## 记录字段

每条模拟排班记录包含：

- 员工号
- 姓名
- 项目名称
- 培训开始日期
- 培训结束日期
- 备注

开始日期和结束日期必须有效，结束日期不能早于开始日期。

## 判断口径

- 模拟记录等同于项目 sheet 中一条“培训信息是否录入 = 否”的计划记录。
- 备注为空时使用“模拟排班”。
- 备注不含“取消”时，记录作为有效安排参与排班覆盖判断。
- 备注包含“取消”时，记录不作为有效安排覆盖人员风险。
- 模拟记录只影响排班总览，不参与有效期更新、CRM 年度核对、Excel 健康检查或更新月份选择。

## 数据边界

- 模拟记录只保存在当前浏览器页面内。
- 导入新的培训 Excel 后，已有模拟记录会被清空。
- 可以删除单条模拟记录，也可以清空全部模拟记录。
- 模拟排班不写回 Excel，不覆盖原文件，也不会自动提交到外部系统。
