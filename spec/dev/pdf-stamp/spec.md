# PDF 加水印开发规格

## 产品边界

`pdf-stamp` 给一份 PDF 的指定页面叠加同一张图片，可建立多条页码、位置、尺寸和透明度规则。页面预览和最终导出都在浏览器本地完成。

## 输入契约

- 必须上传一份可由 PDF.js/PDF-lib 读取的 PDF 和一张浏览器可解码图片。
- 每条规则包含应用模式、页码范围、毫米坐标、毫米尺寸、透明度和锁定宽高比状态。
- 页码范围支持逗号分隔的单页及闭区间，自动去重并按 PDF 总页数过滤。
- 应用模式支持全部页、奇数页、偶数页和指定页。

## 编辑规则

- 新规则按水印图片宽高比生成默认尺寸。
- 位置和尺寸可由表单修改，也可在预览画布上拖动、从控制柄缩放。
- 画布像素与 PDF 毫米坐标双向换算；拖动和缩放限制在页面边界内。
- 锁定比例时修改宽或高会同步调整另一边。
- 多条命中同一页的规则按规则顺序全部叠加。

## 预览与导出

- 普通预览用于编辑当前规则；“导出预览”显示当前页全部命中规则的叠加效果。
- 导出计划只为至少命中一条规则的页面创建水印操作，未命中页原样保留。
- 导出生成新 PDF，不修改源文件；文件名追加水印后缀。
- 缺少 PDF、图片、有效规则或生成失败时禁用导出或显示错误。
