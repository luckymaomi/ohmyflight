# 姓名匹配员工号开发规格

## 产品边界

`crew-match-name-id` 从输入文本中按花名册姓名做包含匹配，输出员工号、分部、技术信息和技术等级。它是浏览器本地工具，不做模糊姓名纠错、别名合并或同名人员判定。

## 输入契约

- 花名册支持 `.xlsx`、`.xls`，默认尝试加载 `public/tool/template/机组花名册.xlsx`，也允许用户重新选择。
- 第一行必须按表头包含 `员工号`、`姓名`、`分部`；`技术信息` 可选。表头匹配会移除空白，并接受包含关键字的表头。
- 员工号或姓名为空的数据行不进入花名册。
- 待查询文本为任意字符串；匹配按花名册中的完整姓名是否出现在文本中判断。

## 技术等级规则

技术等级从 `技术信息` 冒号后的内容中依次识别：`X类`、`飞行教员X`、末尾等级标识。字母统一转为大写，允许字母后带一至两位数字；无法识别时留空。

## 结果与导出

- 页面结果列固定为姓名、员工号、分部、技术信息、技术等级。
- 用户可勾选结果；复制单列或导出时，有勾选则只处理勾选项，无勾选则处理全部。
- Excel 导出使用相同的五列顺序。
- 花名册缺少必需表头时明确报错，不使用固定列号兜底。
