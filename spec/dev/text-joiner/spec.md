# 文本拼接助手开发规格

`text-joiner` 是纯浏览器文本工具，用于清除输入内容中原有的空白和常见分隔符，再按用户指定的输出分隔符重新拼接非空片段。

## 拼接规则

- 换行、空格及 Unicode 标点和符号均视为输入分隔符，连续分隔符只产生一个边界。
- 空片段不进入结果，片段原有顺序保持不变，不去重、不排序。
- 输出分隔符留空时，所有片段零间隔拼接。
- 输出分隔符有值时，按原值连接片段，不修改该分隔符。
- 输入为空或只含分隔符时，输出为空。

## 页面行为

- 输入文本或修改输出分隔符时立即刷新结果和片段数量。
- 页面提供手动拼接、复制结果和清空控件。
- 所有文本只在当前浏览器页面处理，不上传、不保存。
