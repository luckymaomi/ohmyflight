# 航线班次统计开发规格

## 产品边界

`crew-flight-stats` 按排班表工作表统计花名册人员在各航线出现的班次，并保留无法识别的排班单元格供人工复核。工具只统计姓名出现，不推断岗位、航段或飞行小时。

## 输入契约

- 排班表支持 `.xlsx`、`.xls`，用户选择一个或多个工作表参与统计。
- 每个排班工作表第一行为表头；从第二行开始，A 列为航线名称，其余非空单元格为机组文本。
- 花名册从第二行起读取 B 列姓名。页面可加载默认花名册或由用户选择文件。
- 页面另提供文本姓名识别区，按花名册识别粘贴内容，不进入班次统计主结果。

## 统计规则

- 对每个非空机组单元格，按花名册顺序检查完整姓名包含关系。
- 一个单元格命中某姓名一次，就给该姓名在当前航线增加 `1` 班；同一单元格可以命中多人。
- 多个选中工作表的结果合并；航线按首次出现顺序保留。
- 完全没有命中花名册姓名的非空单元格进入未匹配列表，记录工作表、行号、航线和截断后的原文。
- 统计不做模糊匹配；花名册中的包含关系可能造成短姓名误命中，结果需结合未匹配区和业务原表复核。

## 输出契约

- 页面展示人员乘航线矩阵和未匹配内容。
- Excel 导出工作表为 `航线班次统计`，首列为姓名，后续列为航线，单元格为班次数。
- 未选择工作表、文件未就绪或没有可统计数据时不启用统计或导出。
