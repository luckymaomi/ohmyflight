# 提取员工号开发规格

## 产品边界

`crew-extract-id` 是纯浏览器文本工具，从任意文本中提取独立出现的六位数字，去重、升序排列后供复制。它不读取花名册，也不验证六位数字是否为真实员工号。

## 输入与规则

- 输入为页面文本框中的任意字符串。
- 使用 `\b\d{6}\b` 识别候选值；与其他数字连续相连的六位片段不命中。
- 使用集合去重，再按字符串升序排列。
- 空输入或无命中时结果为空，数量为 `0`。

## 输出与交互

- 结果区每行一个员工号，并显示去重后的数量。
- “一键复制”复制结果纯文本；优先使用 Clipboard API，失败时回退到 `execCommand("copy")`。
- “清空输入”同时清空输入、结果和数量。
- 所有处理在本地页面完成，不产生下载文件。

## 代码与验证

- 页面：`public/tool/app/crew-extract-id/index.html`
- 入口逻辑：`src/tool/app/crew-extract-id/main.ts`
- 当前没有独立规则测试；改动提取正则、排序或复制格式时应补充 `tests/tool/crew-extract-id/` 测试。
