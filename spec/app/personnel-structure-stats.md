# 人员结构统计

## 功能定位

人员结构统计是纯前端工具。用户上传任意人员信息 Excel，工具按表头读取数据，按人员结构报告口径生成统计结果，并支持导出 Excel。

工具不绑定文件名、sheet 名或固定列位置。

## 输入

上传 `.xlsx` 或 `.xls` 文件。

默认读取用户选择的工作表。工具按表头识别字段。

必要表头：

- `姓名`
- `技术信息`
- `RAMA`
- `REUO`
- `RWAS`
- `EAMA`
- `EEUO`
- `EWAS`
- `原单位`
- `检查员资格`
- `行政职务`

可选表头：

- `员工号`
- `RSEA`
- `ESEA`
- `RANC`
- `RORD`
- `RJFK`
- `RLAX`
- `RNLU`

## 代码含义

单飞资格：

- `RAMA`：北美区域
- `REUO`：除俄罗斯外的欧洲区域
- `RWAS`：西亚和撒哈拉以北的非洲区域
- `RSEA`：东南亚及港澳台区域

英语通信/报务资格：

- `EAMA`：北美区域
- `EEUO`：除俄罗斯外的欧洲区域
- `EWAS`：西亚和撒哈拉以北的非洲区域
- `ESEA`：东南亚及港澳台区域

特殊/关注机场：

- `RANC`：安克雷奇
- `RORD`：芝加哥
- `RJFK`：纽约肯尼迪
- `RLAX`：洛杉矶
- `RNLU`：圣卢凯

当前报告结构里的航线资格和报务分类只直接使用美、欧、西亚三类组合：

- 单飞资格使用 `RAMA / REUO / RWAS`
- 报务资格使用 `EAMA / EEUO / EWAS`

## 技术信息规则

`技术信息` 冒号后的内容作为技术身份。

- 包含 `飞行教员`：教员
- 包含 `机长`，且不含 `飞行教员`、不含 `划转`：普通机长
- `划转机长`：在训机长
- 包含 `副驾驶`，且不含 `划转`：普通副驾驶
- `划转副驾驶`：转机型副驾驶

## 总人数和空地人员

- 空勤人员（已注册人员）：教员、普通机长、普通副驾驶。
- 地面人员：固定按 29 人统计。
- 总人数：空勤人员（已注册人员）加固定地面人员 29 人。

划转机长和划转副驾驶不属于已注册空勤人员，但在对应结构表里作为在训机长、转机型副驾驶统计。

## 管理人员

按 `行政职务` 判断：

- 管理人员：`行政职务` 非空
- 非管理人员：`行政职务` 为空

母数为空勤人员（已注册人员）。

## 教员、机长、副驾驶

按空勤人员（已注册人员）统计：

- 教员：飞行教员
- 机长：普通机长，不含划转机长
- 副驾驶：普通副驾驶，不含划转副驾驶

## 机长含以上各级别

母数为教员、普通机长、在训机长。

- 检查员：`检查员资格` 为 `公司检查员` 或 `委任代表`
- C/B/A 类教员：按 `技术信息`
- D/C/B/Z 类机长：按 `技术信息`
- 在训机长：`划转机长`

检查员是资格统计，不参与等级分类求和；等级分类本身应与母数闭合。

## 机长航线资格

母数为教员和普通机长，不含在训机长。

航线资格按 `RAMA / REUO / RWAS` 组合分类：

- 美+欧+西亚
- 美+欧
- 美+西亚
- 欧+西亚
- 仅北美带队
- 仅欧洲带队
- 仅西亚带队

航线机长：

- B 类及以上
- 没有 `RAMA / REUO / RWAS`
- 不是 Z 类机长

左座带飞：

- Z 类机长

## 机长报务

母数为教员和普通机长，不含在训机长。

报务按 `EAMA / EEUO / EWAS` 组合分类：

- 美+欧+西亚
- 美+欧
- 美+西亚
- 欧+西亚
- 单美洲报务
- 单欧洲报务
- 单西亚报务
- 无报务

## 副驾驶级别

母数为普通副驾驶和转机型副驾驶。

- D/C/B 类副驾驶：按 `技术信息`
- A 类副驾驶：`A1类副驾驶` 或 `A2类副驾驶`
- 转机型副驾驶：`划转副驾驶`

## 副驾驶报务

母数为普通副驾驶，不含转机型副驾驶。

报务按 `EAMA / EEUO / EWAS` 组合分类：

- 美+欧+西亚
- 美+欧
- 美+西亚
- 欧+西亚
- 单美洲报务
- 单欧洲报务
- 单西亚报务
- 无报务

## 人员居住情况

按 `原单位` 判断：

- 本地：`原单位` 以 `总队` 开头，或等于 `777返聘`
- 异地：其他原单位

机长居住母数为教员和普通机长，不含在训机长。

副驾驶居住母数为普通副驾驶，不含转机型副驾驶。

## 原单位情况

母数为空勤人员（已注册人员）。

映射规则：

- `总队777`、`777返聘` -> `飞行/总队 777`
- `总队737` -> `飞行/总队 737`
- `总队320` -> `飞行/总队 320`
- `总队909` -> `飞行/总队 909`
- `湖南分公司` -> `湖南`
- `湖北分公司` -> `湖北`
- `新疆分公司`、`新疆分公司（借）` -> `新疆`
- `河南分公司` -> `河南`
- `西安分公司` -> `西安`
- `重庆航空` -> `重庆`
- `汕头分公司` -> `汕头`
- `珠海分公司` -> `珠海`
- `广西分公司` -> `广西`
- `海南分公司` -> `海南`
- `上海分公司（借）` -> `上海`

未知原单位进入未识别数据，不静默丢弃。

## 输出

页面展示：

- 统计摘要
- 按人员结构报告顺序排列的统计表
- 需要核对的未识别数据

导出 Excel：

- `统计结果`
- `规则说明`
- `未识别数据`

输出不包含月份列，也不计算本月变化。工具只反映当前上传表的真实统计结果。
