# 姓名匹配员工号

## 功能

从一段包含人名的文本里识别姓名，并到花名册里匹配对应员工号。

## 运行方式

- 启动站点后打开 `tool/app/crew-match-name-id/index.html`
- 页面会优先自动加载默认花名册 `template/机组花名册.xlsx`
- 也可以手动换成其他花名册文件

## 输入

- 1 份员工花名册 Excel
  至少需要“员工号”“姓名”列
  可选“技术信息”列
- 1 段待识别文本

## 输出

- 匹配结果表
  包含姓名、员工号、技术信息、技术等级四列
- 可分别复制“姓名列”“员工号列”“技术信息列”“技术等级列”
- 技术等级从技术信息自动提取（例如 `777:飞行教员C` 提取为 `C`）

## 输入输出样例

输入样例：

```text
明天请通知张三、李四、王五参加复训。
```

花名册样例：

```text
123456 张三
654321 李四
222333 王五
```

输出样例：

```text
张三  123456  777:飞行教员C    C
李四  654321  777:A2类副驾驶   A2
王五  222333  777:E类副驾驶    E
```

## 使用要点

- 适合把聊天记录、通知文本快速转成员工号名单
- 复制时如果勾选了部分结果，就只复制勾选项
- 如果默认花名册加载失败，可以手动重新选择文件
