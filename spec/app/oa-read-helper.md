# 自动点 OA 助手（测试）

## 功能

自动打开 OA 待办页并循环点击“已阅”，适合快速清理大量常规待办。

## 运行方式

- 打开 `tool/app/oa-read-helper/index.html` 查看说明并下载脚本
- 本体运行命令：

```bash
python app.py
```

## 输入

- 模式选择：
  - 无限模式
  - 有限模式
- 有限模式下需要输入点击次数
- 运行前需要人工完成登录

## 输出

- 浏览器中的自动点击动作
- 终端中的处理日志
- 统计本轮点击了多少条、是否有失败或需要人工处理的条目

## 输入输出样例

输入样例：

```text
模式：有限模式
点击次数：20
```

输出样例：

```text
准备处理首条待办：关于值班调整的通知
已点击“已阅”
累计完成：20 条
```

## 使用要点

- 大多数普通 OA 可以自动处理
- 督办等特殊类型可能需要人工点完后再回到脚本继续
- 无限模式适合边工作边挂着跑
