# 培训工具合集

## 功能

这是培训相关工具的合集入口，当前包含 4 个功能页：

- 培训预排班
- 培训有效期更新
- 培训冲突核对
- 飞行部培训规则速查

它保留了“分页面入口”的使用方式，但底层业务口径已经尽量和 `super-training-test` 对齐。

## 运行方式

- 启动站点后打开 `tool/app/training-tools-suite/index.html`
- 再进入对应页面：
  - `schedule.html`
  - `validity-update.html`
  - `conflict-check.html`
  - `rule-reference.html`

## 1. 培训预排班

### 输入

- 规则表文件
- 有效期表文件
- 项目
- 预排开始日期
- 预排结束日期

### 输出

- 页面上的分类名单预览
- 导出 Excel
- 常见输出工作表：
  - `摘要`
  - `已过期`
  - `命中窗口`
  - `本阶段到期`
  - `缺少旧有效期`
  - `未命中窗口`

### 样例

```text
项目：危险品
预排区间：2026-04-01 至 2026-04-30
输出：张三进入“命中窗口”
```

## 2. 培训有效期更新

### 输入

- 1 份“规则表 + 某个培训表”的组合文件
- 1 份有效期表文件

### 输出

- 页面上的更新预览
- 导出 Excel
- 常见输出工作表：
  - `摘要`
  - `更新预览`
  - `跳过记录`
  - `更新后有效期表`

### 样例

```text
培训表中：
123456 张三 危险品 培训信息是否录入=是

输出：
张三 新有效期计算成功，进入“已更新”
```

## 3. 培训冲突核对

### 输入

- 1 份培训表文件
- 至少能识别：
  - 姓名
  - 项目名称
  - 培训开始日期
  - 培训结束日期

### 输出

- 冲突预览
- 导出 Excel
- 导出时会把冲突行整行标红

### 样例

```text
张三：
CRM  2026-04-10 至 2026-04-12
危险品  2026-04-12 至 2026-04-14

输出：
首尾同日，判定冲突
```

## 4. 飞行部培训规则速查

### 输入

- 无需上传文件也可直接查看
- 需要模板时可点击下载

### 输出

- 规则口径说明
- 项目对照表
- 案例说明
- 字段规范
- 模板下载入口

### 样例

```text
查询“危险品”
可看到有效期、窗口口径和项目对照信息
```

## 使用要点

- 如果你想分页面做单项工作，用它更顺手
- 如果你想直接处理新总表，优先考虑 `super-training-test`
- 规则页本身不参与计算，但它负责统一说明口径和模板
