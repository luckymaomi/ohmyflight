# 历史航班统计

## 功能

从任务记录里筛出指定航班号，并把“匹配航班”“每个机长执行某航班的最后一班”“每个机长最后执飞的匹配航班”做颜色标记。

## 运行方式

- 启动站点后打开 `tool/app/flight-filter/index.html`
- 这是浏览器本地工具

## 输入

- 1 份任务导出 Excel
- 目标航班号
  多个航班号可用逗号分隔
- 统计截止日期

## 输出

- 页面上的统计结果预览
- 1 份标色后的 Excel
- 如有未命中记录，还会附带“无记录说明”工作表

## 输入输出样例

输入样例：

```text
目标航班号：CZ3101,CZ3103
截止日期：2026-04-15
```

输出样例：

```text
黄色：所有 CZ3101 / CZ3103 记录
红色：每个机长执行该航班的最后一班
蓝色：每个机长在全部匹配记录中的最后执飞
```

## 使用要点

- 适合做历史航班履历回看
- 颜色本身就是结果的一部分，导出后最直观
- 原始任务表至少要能识别员工号、姓名、日期、任务这几列
