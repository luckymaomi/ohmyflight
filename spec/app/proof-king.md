# 校对之王

## 功能

用于对同一本手册的新旧版本做本地文字比对，并导出 Excel 差异报告。

工具只做文本差异提示和候选定位，不做业务结论判断。

## 运行方式

- 启动站点后打开 `tool/app/proof-king/index.html`
- 页面在浏览器本地读取文件、比对和导出，不上传到服务器

## 输入

- 基准手册 A
  旧版或基准版手册
- 待校对手册 B
  新版或待校对版手册
- 支持格式
  `.docx` 和文字型 `.pdf`
- PDF 页码范围
  PDF 可以填写起始页和结束页，只比对指定页码范围
- Word 范围
  Word 当前按全文读取，不支持页码范围

## 比对规则

- A/B 有主次
  A 是基准，B 是待校对。
- 主报告按 A 的切片顺序输出
  每条 A 切片在 B 中寻找最佳候选。
- B 新增内容单独输出
  系统反向用 B 切片在 A 中寻找对应，找不到可靠对应的内容进入 B 新增。
- 切片优先使用标点边界
  强边界包括换行、句号、分号、问号、感叹号。
- 长句会继续拆分
  逗号、顿号、冒号用于过长内容的二级拆分；仍过长时使用滑动窗口。
- 太短且没有关键项的片段不作为强判断依据
  工具默认不内置业务词特判；需要排除词或增强词时，应通过配置池或 hook 注入。
- 匹配前会做归一化
  统一全角半角、大小写、空白和常见标点影响。
- 归一化保留业务关键项
  数字、英文和条款号会保留，例如 `ICAO`、`1000小时`、`121.467`。
- 高频重复短片段会按页眉页脚类噪音过滤
  过滤只看同一文档内短片段重复次数和长度，不内置具体业务词。
- 搜索召回只用于找候选
  工具使用本地 FlexSearch 和 ngram 候选索引召回相似片段，再按覆盖率、相似度和关键项匹配情况评分。
- 疑似冲突
  两边文本相似，但关键数字、英文或条款号不一致时，进入疑似冲突。

## 状态说明

- 一致
  A 切片在 B 中找到高度一致内容。
- 修改
  A 切片在 B 中找到相似内容，但文字、数字、英文或条款号存在变化。
- 删除
  A 切片在 B 中没有找到可靠对应。
- 需确认
  找到弱候选，但相似度不足，需要人工判断。
- B 新增
  B 切片在 A 中没有找到可靠对应。

## 输出

- 页面统计
  显示 A 覆盖率、一致、修改、删除、B 新增、疑似冲突数量。
- 差异预览
  页面左侧显示重点差异列表，右侧显示选中差异的 A/B 完整上下文。
- 颜色高亮
  页面用颜色标出当前命中的原文、删除内容、新增内容和关键项变化，便于人工直观看差异。
- PDF 页面预览
  B 手册是 PDF 时，点击差异后会预览对应 PDF 页面。
- Excel 差异报告
  包含 `总览`、`A到B比对`、`B新增内容`、`疑似冲突` 四个 sheet。

## Excel 结构

`总览` sheet：

- 基准手册 A
- 待校对手册 B
- A 切片数
- B 切片数
- 一致
- 修改
- 删除
- 需确认
- B 新增
- 疑似冲突
- A 覆盖率
- 平均相似度

`A到B比对`、`B新增内容`、`疑似冲突` sheet：

- 序号
- 状态
- A 或 B 位置
- 标题
- 原文
- 匹配率
- 覆盖率
- 对应位置
- 对应原文
- 缺失关键项
- 新增关键项
- 说明

## 禁止行为

- 不做 OCR
- 不支持扫描件 PDF
- 不支持 `.doc` 旧 Word
- 不使用 AI 或联网语义判断
- 不自动认定业务符合或不符合
- 不写回 Word 或 PDF
- 不把结果自动提交到任何外部系统

## 使用要点

- A 应选择旧版或基准版。
- B 应选择新版或待校对版。
- PDF 页码范围为空时默认读取整本 PDF。
- Word 页码由排版决定，浏览器读取纯文本时没有稳定页码，因此当前不支持 Word 页码范围。
- 差异报告用于人工复核；匹配率高不等于业务含义完全一致。
