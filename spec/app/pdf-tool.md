# PDF 工具箱

## 功能

这是一个多功能 PDF 工具，当前包含 4 个常用模式：

- 页面提取
- PDF 合并
- PDF 转图片
- 图片转 PDF

## 运行方式

- 启动站点后打开 `tool/app/pdf-tool/index.html`
- 这是浏览器本地工具，不上传文件

## 输入

### 页面提取

- 1 份或多份 PDF
- 选中的页码范围

### PDF 合并

- 多份 PDF
- 可手动拖拽调整顺序

### PDF 转图片

- 1 份或多份 PDF
- 图片格式和缩放倍率

### 图片转 PDF

- 1 张或多张图片
- 可手动拖拽调整顺序

## 输出

### 页面提取

- 提取后的 PDF

### PDF 合并

- 合并后的单个 PDF

### PDF 转图片

- 图片文件
- 多个 PDF 时通常打包为 ZIP

### 图片转 PDF

- 由图片生成的 PDF

## 输入输出样例

输入样例：

```text
文件 A：培训手册.pdf
操作：提取第 1-3 页、第 8 页
```

输出样例：

```text
培训手册_提取.pdf
内容：共 4 页
```

## 使用要点

- 页面提取支持多文件分别处理
- 合并和图片转 PDF 都支持拖拽排序
- PDF 转图片适合做截图素材，页面提取适合拆分手册
