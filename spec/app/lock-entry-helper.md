# 通用锁班助手

## 功能

自动填写飞行门户的非生产任务录入表单，支持批量锁班、手动单条录入和 Excel 导入。

## 运行方式

- 打开 `tool/app/lock-entry-helper/index.html` 查看说明、下载脚本和模板
- 本体运行命令：

```bash
python app.py
```

## 输入

- 批量粘贴文本，或 Excel 模板
- 每条记录至少包含：
  - 员工号
  - 姓名
  - 锁班类型
  - 开始日期
  - 结束日期
- 可选输入：
  - 白名单员工号
  - 统一备注

## 输出

- 自动提交到飞行门户
- 终端里的成功 / 失败提示
- 批量粘贴和 Excel 导入都会新建结果 Excel
- 每处理一条记录就实时写入一行结果，不回写原始 Excel
- 成功记录备注为空
- 冲突或错误记录会写入备注；冲突记录会写入门户冲突列表里的“冲突”列内容
- 结束后仍会生成失败记录文本日志，便于复制重跑失败项

## 输入输出样例

输入样例：

```text
123456 张三 ALV 2026-04-20 2026-04-22
654321 李四 MAT_FA_LVE 2026-04-21 2026-04-21
```

输出样例：

```text
张三：录入成功
李四：发现冲突，结果 Excel 写入冲突原因，已跳过
结果 Excel 和失败记录日志已保存到本地
```

## 使用要点

- 启动后先登录门户，再开始批量处理
- 统一备注确认一次后，可复用于本轮全部记录
- 遇到冲突不会强行提交，适合批量安全处理
