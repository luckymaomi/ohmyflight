# 提取员工号

## 功能

从一段混杂文本里提取所有 6 位数字员工号，自动去重、排序，并支持一键复制。

## 运行方式

- 启动站点后打开 `tool/app/crew-extract-id/index.html`
- 这是浏览器本地工具，不需要安装 Python

## 输入

- 任意文本
- 文本里只要包含 6 位数字，就会被当作候选员工号

## 输出

- 去重后的员工号列表
- 页面上的数量统计
- 可直接复制的纯文本结果

## 输入输出样例

输入样例：

```text
今天值班：123456 张三，备用 654321 李四。
另外请联系 123456 和 222333。
```

输出样例：

```text
123456
222333
654321
```

## 使用要点

- 只提取 6 位数字
- 同一个员工号重复出现只保留 1 次
- 结果更适合直接粘贴到 Excel、OA 或聊天窗口里
