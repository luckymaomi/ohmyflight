# 飞行经历 / 左座经历起落数按天统计（测试）

## 功能

自动登录飞行门户，按员工和日期区间批量查询“飞行经历”或“左座经历”，并把结果整理成 Excel。

## 运行方式

- 打开 `tool/app/flight-stats-helper/index.html` 查看说明并下载脚本
- 本体运行命令：

```bash
python app.py
```

## 输入

- 查询类型
  - 飞行经历
  - 左座经历
- 输入方式
  - Excel 模板导入
  - 直接粘贴文本
- 每条记录至少包含：
  - 员工号
  - 姓名
  - 开始日期
  - 结束日期

## 输出

- 1 个 Excel 结果文件
- 结果列通常包含：
  - 员工号
  - 姓名
  - 起始时间
  - 截止时间
  - 飞行经历或左座经历
  - 起落总数

## 输入输出样例

输入样例：

```text
123456 张三 2026-04-01 2026-04-10
654321 李四 2026-04-01 2026-04-10
```

输出样例：

```text
员工号    姓名  起始时间      截止时间      飞行经历  起落总数
123456   张三  2026/04/01   2026/04/10   32:15     8
654321   李四  2026/04/01   2026/04/10   18:40     4
```

## 使用要点

- 粘贴模式和 Excel 模式都支持批量查询
- 更适合一次查很多人，而不是临时查 1 条
- 查询完成后结果会直接落到本地 Excel 文件
