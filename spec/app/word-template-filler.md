# Word 模板填充器

## 功能

把“配置表 + Word 模板”打包成一个可独立分发的小应用，让别人按表单填写后再导出 Word；生成的小应用也支持通过 Excel 批量导入信息并批量生成 Word。

## 运行方式

- 启动站点后打开 `tool/app/word-template-filler/index.html`
- 这是浏览器本地工具

## 输入

- 1 份 Excel 配置文件
  用来定义字段名、显示名称、类型、选项、默认值等
- 1 份 `.docx` 模板
  模板里使用占位符，例如 `{name}`、`{date}`
- 1 个应用名称

## 输出

- 1 个 ZIP 包
- ZIP 内通常包含：
  - 可直接打开的 HTML 填写页面
  - 原始 Word 模板
  - 批量导入 Excel 模板（配置不含 `列表/循环` 字段时）
  - 离线运行所需依赖
  - 部署或使用说明
- 生成的小应用支持：
  - 单份填写并导出 1 份 Word
  - 导入 ZIP 包内随包提供的批量 Excel 模板填写结果，并预览行数和错误
  - 批量导出 Word ZIP
  - 批量 Excel 第一列为 `文件标题`，用于控制每份 Word 的文件名；为空时自动使用“应用名 + 序号”

## 批量导入限制

- 第一版批量导入暂不支持 `列表/循环` 字段
- 如果配置里包含 `列表/循环` 字段，生成的小应用会禁用批量导入区域并提示暂不支持
- 单份填写仍支持 `列表/循环` 字段

## 输入输出样例

输入样例：

```text
配置字段：
name  姓名  文本
date  日期  日期

Word 模板：
姓名：{name}
日期：{date}
```

输出样例：

```text
生成一个“请假申请表”小应用
用户填写姓名和日期后，可导出对应 Word
```

## 使用要点

- 适合做重复使用的申请单、登记表、报告单
- 配置模板可以直接从页面下载，说明和示例写在配置模板内
- 重点不是马上导出 1 份 Word，而是先生成一个以后还能反复用的小工具
- 如果已经用 Excel 收集好多人信息，优先使用 ZIP 包里随包导出的批量导入模板和生成应用里的批量导入功能
