# Word 模板填充器

## 功能

把“配置表 + Word 模板”打包成一个可独立分发的小应用，让别人按表单填写后再导出 Word。

## 运行方式

- 启动站点后打开 `tool/app/word-template-filler/index.html`
- 这是浏览器本地工具

## 输入

- 1 份 Excel 配置文件
  用来定义字段名、显示名称、类型、选项、默认值等
- 1 份 `.docx` 模板
  模板里使用占位符，例如 `{name}`、`{date}`
- 1 个应用名称

## 输出

- 1 个 ZIP 包
- ZIP 内通常包含：
  - 可直接打开的 HTML 填写页面
  - 原始 Word 模板
  - 部署或使用说明

## 输入输出样例

输入样例：

```text
配置字段：
name  姓名  文本
date  日期  日期

Word 模板：
姓名：{name}
日期：{date}
```

输出样例：

```text
生成一个“请假申请表”小应用
用户填写姓名和日期后，可导出对应 Word
```

## 使用要点

- 适合做重复使用的申请单、登记表、报告单
- 配置模板和示例文件可以直接从页面下载
- 重点不是马上导出 1 份 Word，而是先生成一个以后还能反复用的小工具
