# 文件批量处理工具

## 功能

批量重命名文件、修改扩展名、创建同名文件夹并移动文件，适合大批量整理本地文件。

## 运行方式

- 打开 `tool/app/file-batch-tool/index.html` 查看说明并下载脚本
- 本体运行命令：

```bash
python file_batch_tool.py
```

## 输入

- 1 个本地文件夹
- 1 种处理动作
  例如：
  - 添加前缀
  - 添加后缀
  - 删除字符
  - 替换字符
  - 修改扩展名
  - 创建同名文件夹

## 输出

- 预览后的新文件名
- 执行后的实际重命名结果
- 最近一次操作支持撤销

## 输入输出样例

输入样例：

```text
文件夹内：
photo.jpg
report.jpg

动作：
添加前缀 2026_
```

输出样例：

```text
2026_photo.jpg
2026_report.jpg
```

## 使用要点

- 先预览，再执行
- 修改扩展名只是改名字，不会真的转换文件格式
- 重要文件建议先备份
