# 飞行人员训练大纲阅读

## 手册定位

《飞行人员训练大纲》规定各类训练如何实施：谁可以进入、使用什么设备和资料、完成哪些理论与实践课程、由谁考试、合格后进入哪一阶段、资格如何保持或恢复，以及训练记录如何保存。

回答训练问题时，以完整课程链为单位读取：适用对象与进入条件、理论、固定模拟机、全动模拟机、本场训练、航线运行经历、检查、资格结果和记录。完成课程不自动等于技术等级变更、岗位聘任或运行资格发布；这些管理动作分别回到《飞行技术管理手册》和《运行手册》核对。带 `*` 且正文明确留空的章节，只记录为现行样本无课程内容。

## 证据规则

- 用户提供的现行大纲原文是唯一最终依据。本 Skill 只负责告诉你完整地去哪里看、如何串联。
- 必须读取信息页、有效条款清单、全部50章、所有表格和附录；不能只读与当前问题同名的一章。
- 章节标题带 `*` 时，先检查正文是否明确“有意留空”。目录中出现名称不代表公司当前实施该课程。
- 训练课时、课数、座位、PF/PM角色、起落、航段、考试、检查和资格发布是不同环节，不得合并成一个“已完成”。
- 进入条件通常同时引用执照、原机型经历、安全记录、英语等级、技术等级和其他手册。必须交叉核对。
- Word表格中的课程安排、最低起落、评分尺度、循环科目和设备清单属于正文证据，不能转成无结构文本后忽略列关系。

## 与另外两本手册的边界

- 本大纲回答“具体怎么训练”。
- 《飞行技术管理手册》回答“谁能进入、检查后获得什么技术等级、如何聘任和管理”。
- 《运行手册》回答“资格在运行中如何使用、近期经历、搭配和值勤限制”。
- 某项训练合格不自动等于岗位聘任或运行资格已发布。需要分别核对技术管理流程和运行权限。

## 每章通用阅读模板

除明确留空章节外，每个训练提纲至少按以下顺序读取：

1. 训练目的和适用范围：确认课程解决什么问题、适用于哪类人员。
2. 训练设备和设施：确认允许使用的模拟机、训练器、教室或其他设备。
3. 训练资料：确认规章、厂家资料、公司手册和课件依据。
4. 教学方法：确认课堂、CBT、远程、固定模拟机、全动模拟机、飞机或航线训练方式。
5. 进入条件：逐项核对执照、岗位、技术等级、经历、英语、安全记录和前置资格。
6. 课程安排：保留理论小时、课数、模拟机小时、座位、PF/PM、起落和航段结构。
7. 考试/评估：确认推荐、考试员、合格标准、不合格处理和完成期限。
8. 资格获取与保持：确认训练合格后是否还需要运行经历、航线检查、审批或系统发布。
9. 过期和恢复：确认失效起算点、分档课程、重新检查和降级处理。
10. 训练记录保存：确认记录内容、责任单位和保存要求。

## 人员成长主线

### 新雇员到副驾驶

按第2章确认新雇员来源和基础课程，再按第5章读取副驾驶转机型全流程。重点核对：

- 不同进入条件是否对应不同理论课时和前置经历。
- 理论考试合格后进入模拟机的期限，超期后的补充学习。
- 固定模拟机、全动模拟机、LOFT、本场训练的课数、小时和座位。
- 本场训练结束后监督飞行、航段、PF、人工飞行和航线检查。
- 训练结束并不直接证明A1/A2/B/C/D等级，等级变化要回到技术管理手册。

### 机长阶段

第7章主要是已有机长转入新机型，不等于副驾驶升级训练。阅读时核对：

- 原机型技术等级、机长经历、总驾驶员经历和优先条件。
- 左座模拟机、本场训练、右座观察、左座航线经历和夜航要求。
- 航线检查的起始条件、完成上限和不合格处理在技术管理手册中的补充规定。
- 当前第8章留空，因此副驾驶升机长问题必须查现行技术管理手册和其他适用文件，不能用空章补造流程。

### 教员和检查员

- 第9章查航线教员的初次申请、原教员转入、带飞能力、更新和恢复。
- 第10章查型别教员的教学能力、面板操作、实践考试、等级签注和专职模拟机教员。
- 第12章查飞行检查员训练；聘任、权限、工作量、年度考核和任期还要查技术管理手册附录B。

### 资格失效和恢复

第14章不能只看“失去几个月”，要先判断失效原因：

- 未按期完成复训或熟练检查。
- 90日近期经历不足。
- 新机型/新职位后未在规定时间完成航线经历。
- 熟练检查、航线检查或重新获得资格实践考试不合格。
- 教员等级、运行资格或其他专项资格过期。

不同原因对应不同课程、课数、航线监视、检查和降级结果。

## 关键规则卡

以下规则卡说明当前研究样本中关键数字所属的课程阶段、适用对象和未满足后的处理。实际安排训练前必须核对现行大纲全文和人员进入条件。

### 新雇员训练的140小时和40小时

- 新雇员不是统一上一套课程。飞行院校毕业人员与来自其他公司、军转等人员对应不同进入条件。
- 当前样本中，两类人员的地面理论总量分别出现140小时和40小时；差异来自既有经历和需要补齐的公司运行知识。
- 课程还可能包含应急生存、差异训练和疲劳管理，不能只完成地面理论总时数就结束新雇员流程。
- 同型别转入人员是否使用缩减复训课程，需要批准，不能由单位自行按40小时执行。

### 副驾驶转机型是一条连续训练链

1. 先确认执照、总飞行经历、原机型和运行经历满足进入条件。
2. 完成规定机型理论，当前样本基线为140小时，考试达到规定合格分数。
3. 在理论训练后规定期限内进入飞行训练器或模拟机；超过期限，需要补充理论学习和考试。
4. 完成固定模拟机和全动模拟机课程。当前样本出现固定模拟机4课16小时、全动模拟机12课48小时（含LOFT和考试）。
5. 在右座完成本场模拟机训练。当前样本要求两人一组2课8小时，每人达到最低起落数，并在模拟机训练结束后规定期限内完成。
6. 在检查员或教员监督下建立至少25小时航线经历，包含至少4次飞行、至少3次作为操作驾驶员，并有一次在10000英尺以下人工飞行。
7. 完成训练不直接等于A2或更高等级；A1/A2/B/C/D变化继续查技术管理手册。

### 机长转机型的左座和右座

- 这章适用于已有机长进入新机型，不是副驾驶升级训练。
- 训练前先核对原机型机长等级、机长经历和总驾驶员经历。
- 当前样本同样包含理论、固定模拟机和全动模拟机，但本场训练在左座完成。
- 航线阶段先有右座飞行/观察要求，再在左座作为操纵驾驶员完成规定航段和小时，其中包含夜航和10000英尺以下人工飞行要求。
- 左座经历完成后还要按技术管理手册实施机长航线检查、聘任和等级管理。

### 80分合格线和60天窗口

- 总则当前样本规定，地面理论课程结束后考试达到80分（含）为合格，由认可的训练机构组织并写入飞行员电子记录系统；考试不合格按《飞行技术管理手册》处理。
- 副驾驶和机长转机型课程中的本场模拟机训练，应在前一阶段模拟机训练结束后60天内完成。该窗口的起点是模拟机训练结束日，终点是本场模拟机训练完成日。
- 数据链通信课程的操作训练，应在地面理论训练完成后60天内实施；UPRT等专项课程另有“理论结束后60天内开始模拟机训练”的要求。
- 记录60天窗口时同时写明课程名称、起算事件、起算日期、要求完成的环节和实际完成日期。不同课程出现相同数字，不合并成一个通用有效期。

### 25小时和120天100小时是两层要求

- 转机型后的基础运行经历至少25小时，并包括至少4次飞行、至少3次作为操作驾驶员，其中一次在10000英尺以下人工飞行；这是课程进入航线检查前的运行经历要求。
- 技术管理手册对A1转A2另以25小时、4个起落和24分钟人工飞行时间管理等级升级，办理时分别核对飞行次数、起落和人工飞行证据。
- 120天内100小时用于新机型或新职位后的持续航线经历要求。
- 未在120天内完成100小时，通常需要熟练检查并重新建立该时间窗口；这不是把剩余小时继续累加即可。

### 航线教员和型别教员

- 航线教员先检查安全记录、原教员经历、机长小时和带飞起落，再进入基础理论、机型理论、在座、本场和带飞能力检查。
- 型别教员在航线教员基础上增加机长经历、模拟机面板操作、教学训练和实践考试。
- `36个月有效、到期前90天更新` 描述资格保持窗口，不代表任何时候都能申请；对应机长资格和熟练检查也必须有效。
- 失去教员资格后，通常要先恢复机长资格并建立规定机长经历，再完成理论、在座、面板或实践考试。

### 重新获得资格先判断“为什么失去”

| 原因/时长 | 阅读方式 |
| --- | --- |
| 未按期复训或熟练检查 | 按失去不超过6个月、7至12个月、13至24个月、24个月以上分档查模拟机课数和航线监视要求。 |
| 90日内不足3次起飞着陆 | 查近期经历重新建立课程；如果不能结合复训完成，可能需要模拟机专场。 |
| 120天内未完成100小时 | 完成熟练检查并重新建立120天100小时，不按普通“补剩余小时”处理。 |
| 实践考试或熟练检查不合格 | 查6小时、8小时补充训练，以及增加200小时副驾驶经历、降为D类副驾驶、停飞或再次升级的完整后果链。 |

### 专项运行资格

- LVO：机长和副驾驶进入条件不同；理论、PF/PM模拟机训练、签注周期和超过一定时长后的重新获取要分别读取。
- 特殊机场：机长/副驾驶机型经历不同；理论或模拟机只是前置，运行经历和航线检查决定资格获取；长期没有起降经历会失效。
- EDTO：地面理论、模拟机和EDTO航段经历共同构成获取过程；检查不合格要增加航段后再检，取得后按周期复训。
- 高原：先区分一般高原和高高原，再核对机长/副驾驶经历、理论、模拟机和运行经历。
- 英语通信：初次区域资格和增加其他区域资格所需航段不同；检查不合格后的再次经历也有上限。

### EBT评分上下文

- EBT不是只看一次分数，而是半年3课12小时的连续训练和评估结构。
- 1分表示未达到最低可接受水平，进入运行前必须补充训练并达到最低要求。
- 2分可能允许运行，但连续出现、同次训练多项出现时会触发针对性训练或规定期限内补充训练。
- 判断时要看哪一课、哪个胜任力、连续几次复训和补充训练结果，不能看到“2分”就统一停飞。
