# 飞行技术管理手册阅读

## 手册定位

《飞行技术管理手册》规定飞行人员技术资格的管理主线：进入训练的条件、训练和检查的组织、技术等级和运行资格的取得、岗位聘任、资格保持与恢复、技术档案和资质发布。

处理人员状态时分别确认以下对象，不能用其中一项替代另一项：

- **训练完成**：说明规定课程和检查已完成，不必然代表岗位或资质已经发布。
- **技术等级**：例如A1/A2/B/C/D类副驾驶、不同类别机长，反映人员所处技术阶段。
- **运行资格**：例如区域、航线、机场、EDTO、LVO等，决定能否执行特定运行。
- **岗位聘任**：公司正式安排其担任副驾驶、机长、教员或考试员，是独立管理动作。
- **执照/签注**：局方证照层面的权利，不能被公司技术等级或培训记录替代。
- **检查结论**：是进入下一阶段的条件之一；还可能需要经历、审批、聘任和系统发布。

## 证据规则

- 把用户提供的现行手册原文作为唯一最终依据。本 Skill 是阅读地图，不替代手册。
- 必须读取完整文件，包括信息页、有效条款清单、目录、正文、表格、附件和附录；不得只搜索几个关键词后声称读完。
- 表格中的进入条件、评分、扣分和流程与正文同等重要。只提取普通段落会漏掉关键规则。
- 章节号、责任单位、人员等级、小时数、起落数、航段数、月份、有效期和处理结果都可能修订。回答具体问题时重新核对现行原文。
- 区分“技术等级”“运行资格”“执照签注”“岗位聘任”“检查结论”和“训练完成”。这些概念可能相关，但不能互相替代。
- 对带条件、例外、经批准偏离、原机型经历折算和不合格处理的条款，必须连同前后款一起阅读。

## 与另外两本手册的边界

- 本手册回答“能否进入、检查后获得什么、如何管理”。
- 《飞行人员训练大纲》回答“具体训练课程、课时、科目和考试怎么实施”。
- 《运行手册》回答“取得资格后，在真实运行中如何搭组、能否操纵起降、近期经历和值勤限制是什么”。
- 同一数字出现在三本手册时，先判断其角色：运行手册可能规定政策，本手册规定管理流程，训练大纲规定课程和恢复办法。

## 完整阅读顺序

1. 读取信息页、修订记录和有效条款清单，确认版本、生效范围和哪些章节最近变化。
2. 读取目录并建立完整标题树，记录正文与附录边界。
3. 按第1至10章顺序阅读正文，不跳过表格和附件。
4. 按附录A至K逐项阅读；附录中的管理办法是当前手册组成部分，不是可忽略资料。
5. 建立“人员阶段—进入条件—训练—检查—获得资格—保持/失效—恢复—记录”的链路。
6. 对所有数字门槛建立清单，并回到原文核对适用人员、起算点、截止点和例外。

## 人员成长主线

### 新雇员和转机型

优先阅读第3章训练实施，再交叉阅读第6章聘任、第9章档案和训练大纲相应课程。核对：

- 人员来源和原有机型经历属于哪种进入条件。
- 新雇员训练、机型理论、模拟机、本场训练、航线运行经历和航线检查的顺序。
- 理论、模拟机、本场训练和实践考试之间的时间窗口。
- 原机型经历是否允许恢复原技术标准，适用什么额外航线经历和检查。

### 副驾驶等级

在第3章查等级建立条件，在第4章查检查标准，在第5章和运行手册查操纵权限与搭配。当前阅读基线中需要重点搜索：

- `A1`、`A2`、`B类副驾驶`、`C类副驾驶`、`D类副驾驶`、`E类副驾驶`。
- `25小时`、`4个起落`、`人工飞行时间`、`本阶段起落`、`总起落`。
- 一体化课程、综合理论考试、模拟机训练豁免条件、英语通信资格和航线检查。

不要只看到起落数就判断可以升级；课程、考试、英语资格、阶段经历和检查通常需要同时满足。

### 机长阶段

优先阅读第3章机长训练和技术等级、第6章机长聘任、第5章航线资格，再阅读运行手册第4章。重点区分：

- 转机型机长、首聘机长、新聘机长、巡航机长和不同技术等级机长。
- 建立左座经历、航线检查、天气限制、聘任和技术等级晋升不是同一事件。
- `25小时/4个起落`、检查完成上限、`100小时`、`210个日历日`、一年和机长经历门槛的适用对象。
- 不合格后增加经历、退回副驾驶或退回原机型的处理链。

### 教员和考试员

航线教员同时阅读第3章、第6章和附录K；型别教员同时阅读第3章、第6章和附录A；考试员同时阅读第6章和附录B。重点核对：

- 初次申请与原已具备资质人员的不同进入条件。
- 机长经历、带飞经历、起落数、英语等级、安全记录和年龄条件。
- 理论、在座、模拟机教学、观摩、实践考试和年度检查。
- 有效期、到期前更新窗口、权利自动丧失、暂停、终止和重新申请间隔。

| B转C | 完成本阶段一体化课程；B777总起落不少于35个且本阶段不少于5个；通过综合理论考试；完成2小时模拟机训练。若总起落不少于70个且本阶段不少于30个，可按现行条款免除该模拟机训练；同时具备至少两条洲际航线英语通信资格，C类航线检查合格后取得C类资格。 |
| C转D | 完成本阶段一体化课程；B777总起落不少于50个且本阶段不少于5个；通过综合理论考试；完成2小时模拟机训练。若总起落不少于100个且本阶段不少于40个，可按现行条款免除该模拟机训练；同时具备公司在飞全部运行航线的英语通信资格，D类航线检查合格后取得D类资格。 |
| 原机型C类及以上转入 | 转机型后先完成4个航段、25小时运行经历，经航线检查合格取得A2资格；再完成75小时、20个起落和120分钟人工飞行时间，按原技术标准航线检查合格后恢复原机型技术标准，再进入后续阶段训练。 |

### PF、PM、左右座和教员等级

- A1副驾驶通常在右座建立经历，但“坐右座”不等于一定能作为PF。
- L1航线教员与A1搭组时，当前样本规定A1仅作为PM在右座飞行。
- L2航线教员与A1搭组时，A1才可作为PF在右座飞行。
- L1与A2/B/C/D副驾驶搭组时，这些副驾驶可在右座作为PF或PM。
- 型别教员具备L2航线教员教学权限。
- E类副驾驶、Z类巡航机长以及左座权限另有条件，必须完整阅读附录K和运行手册搭配条款，不能沿用A1/A2规则。

办理副驾驶升级时分别留存课程完成、理论考试、起落与人工飞行经历、模拟机训练或豁免依据、英语通信资格和航线检查结果。任何一项缺失，都不能仅凭系统中的技术等级代码反推前置条件已经满足。

### 转机型后的25小时和120天100小时

- 本手册当前样本在A1转A2等级条件中规定至少25小时、4个起落和24分钟人工飞行时间；这些数据用于判断技术等级升级条件。
- 训练大纲当前样本另从课程实施角度规定至少25小时航线经历、至少4次飞行，其中至少3次作为操作驾驶员，并有一次在10000英尺以下人工飞行。办理时分别核对两本手册，不把“4个起落”和“4次飞行”当成同一字段。
- `120天内100小时` 是新机型或新职位训练后的持续航线经历要求，不是对前述25小时的替代。
- 120天内未完成100小时，通常需要重新完成熟练检查并重新建立该时间窗口内的100小时经历。
- 办理时分别记录本场检查日期、航段与起落明细、PF和人工飞行证据、25小时完成日期、100小时累计值和120天截止日期。

### 机长阶段

- 机长转机型后先在右座完成规定观察，再在左座建立至少4个航段、25小时经历；检查还受到最大经历小时和起落上限约束。
- B类机长满足本机型规定机长经历后，经航线技术检查合格转为C类；检查不合格会增加经历后再检，仍不合格可能退回原机型。
- 转C类机长检查存在明确日历日完成期限，超期后果不是简单“延后检查”，要核对现行条款中的降级处理。
- 首聘/新聘机长转更高技术等级还涉及本机型机长年限、机长经历和航线检查。不要把“满一年”“满500小时”拆开使用。
- 当前样本规定转C类机长检查应在第二次检查通过后的210个日历日内完成并通过，逾期结果为降为D类副驾驶；该日期应按二检通过日单独计算，不以排班计划或申请日期代替。
- 机长熟练检查、重新获得资格或专项检查不合格时，可能触发副驾驶补充训练、增加200小时副驾驶经历、再次机长补充训练、暂停机长资格或降为D类副驾驶。应沿第3、4章和训练大纲第14章读取完整处置链，不把某一次补训视为自动恢复。

### 教员和考试员

- 航线教员和型别教员申请通常同时检查机长经历、带飞经历、起落、安全记录、英语和年龄，不是完成一个教员课程就能聘任。
- 教员等级/资格存在以日历月计算的有效期和到期前更新窗口；对应机长资格失效时，教员权利可能自动丧失。
- 考试员的“连续24小时不超过多少检查小时/人次”是履职上限；是否能担任考试员还需满足聘任、近期经历、年度训练、监察和续聘条件。

### 资质发布与档案闭环

1. 先在第3章确认训练和经历完成，在第4章确认检查结论，在第5章确认实际运行权限。
2. 再在第6章确认岗位聘任是否完成；训练合格或检查合格本身不代替聘任。
3. 按附录H确认技术等级、航线/区域/机场及专项运行资格的申请、审核、发布和监督责任。
4. 按第9章与附录I写入技术档案和全生命周期信息，核对生效日期、失效日期、机型、资格代码和对应检查记录。
5. 最后回到《运行手册》确认该资格在具体航班、机组搭配、机场和天气条件下是否可用。

系统中已出现资格代码不代表业务链天然闭合。核查时应能从资格记录追溯到课程、经历、检查、审批/聘任和发布依据；无法追溯的，标记为待核实，不补推缺失事实。

### 岗位胜任力关注

- 高度关注、重点关注、一般关注和预防性关注分别对应不同触发条件、关注期、暂停运行、帮扶和恢复流程。
- “12个月内发生几次事件”必须连同事件类型、是否作风类、评分区间和补充训练结果一起判断。
- 重点关注和一般关注不是同一种停飞处理，帮扶期、航线带飞、检查和重新确定技术等级的流程不同。

### 其他数字主题

还应按上下文核对：考试合格分数和补考次数；理论到模拟机、模拟机到本场、推荐到实践考试的时间窗口；英语通信初次获取和增加区域的航段；长期未在区域运行后的恢复；应急、CRM、PBN和复训周期；档案在离职、退休或退出运行后的保存时间。

## 新版本完整复核

收到新版或其他单位版本时，必须完整执行：

1. 明确旧版和新版方向，记录版本、修订号、生效日期和比较范围。
2. 分别提取完整标题树、正文段落、表格行、附件和附录；验证每章都有内容或明确标记留空。
3. 使用 `compare-manuals` Skill 生成顺序锚点和缺口证据。搜索相似度只用于候选召回，不直接证明一致。
4. 建立逐章清单：新增章节、删除章节、标题移动、正文新增、正文删除、内容修改、表格变化、待确认。
5. 对责任主体、人员等级、数字、单位、时间窗口、否定词、前提条件、例外、处理结果和表格列变化逐项复核。
6. 对第3至6章和附录A、B、G、H、I、J、K做第二轮人工语义复核，因为这些区域直接影响资格和人员工作流。
7. 读取修订记录，但不能只依赖修订记录；以正文和表格真实差异为准。
8. 输出完整原文和位置证据。无法确认对应关系时标记待人工确认，不强行解释为业务等价。
9. 只有确认新版已生效且差异已复核，才更新本 Skill 的章节地图和定位锚点；不得把历史版本继续写成当前事实。
