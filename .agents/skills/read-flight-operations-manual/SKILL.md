# 运行手册阅读

把运行手册理解为公司的“运行总规则”：它规定组织职责、人员和飞机条件、机组搭配、飞行和值勤限制、从准备到着陆的操作政策、签派放行、地面保障、应急和特殊运行。

## 新人先建立这个认识

- 运行手册不是只给飞行员看的操作步骤，它同时规定公司、机长、签派、机务、航站和其他单位的责任。
- “具备资格”只说明人员条件之一；航班还要同时满足飞机、机场、天气、性能、燃油、搭配和值勤条件。
- 计划排班符合不代表实际运行始终符合。延误、备降、置位、备份和延长都可能改变最终值勤数据。
- 机长有紧急处置权不等于可以任意偏离；偏离的触发条件、报告和记录仍要完整执行。
- 表格经常决定最终上限。正文中的一个数字不能替代表格中的报到时刻、航段数、扩编和休息条件。

## 阅读底线

- 用户提供的现行运行手册原文是唯一最终依据。本 Skill 是阅读导航，不替代手册。
- 运行手册篇幅大，必须完整读取信息页、17章正文、表格、检查单、附件和交叉引用，不得只搜索问题关键词。
- 同一主题可能同时出现在总政策、机组政策、飞行程序、特殊运行和附件中。必须沿交叉引用继续读取。
- 表格中的值勤期、飞行时间、天气标准、性能限制、载重数据和报告时限不能脱离表头解释。
- 区分公司政策、机长权力、签派职责、训练要求、人员资质、机场资格、设备批准和实际运行程序。
- 对数字、天气标准、时间窗口、报告时限、例外和“经批准偏离”逐字核对，不凭经验补充。

## 与另外两本手册的边界

- 本手册回答“取得资格后如何运行、受什么限制”。
- 《飞行技术管理手册》回答“资格、技术等级、检查和聘任如何取得与管理”。
- 《飞行人员训练大纲》回答“对应训练课程怎么实施”。
- 运行手册写“必须具备某资格”时，继续查技术管理手册确认资格管理，再查训练大纲确认训练内容。

## 完整阅读顺序

1. 先读信息页、批准页、修订记录、有效条款清单、目录和术语定义。
2. 阅读第1、2章，确认文件体系、组织和职责主体。
3. 阅读第3、4章，建立公司运行政策和机组人员规则主线。
4. 阅读第5、6章，建立从飞行准备到放行、燃油、监控和着陆的运行链。
5. 阅读第7至11章，覆盖适航地面、安保、货运危险品、通信、性能和载重平衡。
6. 阅读第12至14章，覆盖紧急、恶劣天气和特殊运行。
7. 阅读第15至17章，覆盖非计划任务、新航线和航站管理。
8. 汇总所有表格、检查单、报告和其他手册交叉引用，确认没有孤立规则。

## 第3章阅读重点

第3章内容跨度最大，不能把“运行政策”当成一条规则。按问题进入：

- 人员或飞机是否允许飞：3.2至3.7。
- 机场、航路、基地和补充运行：3.8至3.12。
- 检查单、飞行记录本和安全管理：3.13至3.18。
- 机场运行标准、驾驶舱和设备：3.21至3.24。
- 地面滑行、尾流、航线和着陆性能：3.26至3.30。
- 仪表进近、稳定进近和导航数据库：3.31至3.34。
- 驻外、国际运行和特殊飞行程序：3.36至3.38。

进近问题至少同时读取仪表进近、稳定进近、飞行运行下降进近、恶劣天气和低能见度章节。

## 第4章人员规则主线

### 训练、检查和近期经历

重点读取4.1至4.3及4.25：

- 新雇员、转机型、定期复训和熟练检查的基本要求。
- 新机型/新职位后的运行经历和在规定时间内完成航线小时。
- 连续90日内起飞着陆近期经历。
- 机长年度航线检查、驾驶员半年熟练检查及前后月窗口。
- 特殊区域、航线和机场资格及有效期。

### 执照、体检和证件

读取4.4至4.7、4.16和4.19，区分：

- 驾驶员执照有效与公司运行资格有效。
- 体检合格证有效期、年龄差异、提前更新、疾病中断飞行后的鉴定。
- 护照、通行证、飞行任务书和随身用具。

### 机组搭配和操纵权限

读取4.8至4.14：

- 在座资格、机组指派、扩编机组和实力搭配。
- A1/A2/B/C/D/E类副驾驶、不同机长和巡航机长的搭配限制。
- 副驾驶能否操纵起降取决于自身等级、机场类别、机长/教员资格和其他条件。
- 巡航阶段职责替代不等于起飞着陆阶段权限。

### 飞行时间、值勤和休息

完整读取4.15及其表格，核对：

- 计划与实际飞行时间、飞行值勤期、其他值勤、置位和备份如何计入。
- 基本机组与扩编机组限制。
- 日历月、日历年、连续7日等累计限制。
- 值勤之间休息、144小时内连续休息、跨时区恢复。
- 意外延长的机长决定权、报告期限、修正措施和次数限制。

不能只引用一个“最大小时数”；最大值通常由报到时刻、航段数、机组人数、休息设施和延长条件共同决定。

## 运行阶段主链

回答具体航班操作问题时，按时间顺序组织证据：

1. 第6章确认签派放行、天气、计划、燃油和监控。
2. 第5章确认报到、准备、接机、推出和滑行。
3. 第3章确认机场标准、设备、进近、稳定进近和着陆性能政策。
4. 第5章继续确认起飞、巡航、下降、进近、复飞、备降和着陆。
5. 遇到天气读取第13章，专项运行读取第14章，紧急情况读取第12章。
6. 飞行结束后按第1章和相关章节完成记录与报告。

## 关键规则卡

以下用当前研究样本帮助新人理解常见数字的业务上下文。排班或放行前必须读取现行条款、表格和实际人员数据。

### 90天3次起飞着陆

- 适用场景：判断驾驶员能否作为该型别飞机的飞行机组必需成员执行任务。
- 基本含义：在前90个连续日历日内，应在所服务型别完成至少3次起飞和3次着陆；允许使用经批准模拟机的条件要按原文核对。
- 不满足时：不能简单安排一次普通航班补齐，需要按训练大纲规定重新建立近期经历，并满足适用训练和检查要求。
- 核对数据：型别、每次起飞着陆日期、飞机/模拟机来源、座位和是否符合重新建立程序。

### 转机型后的25小时与120天100小时

- 完成新机型或新职位训练后，先在检查员或教员监督下建立至少25小时运行经历，并满足飞行次数、PF和人工飞行要求。
- 之后还要在训练完成后的120天内累计至少100小时航线经历。
- 25小时是初始受监督经历，100小时是规定期限内的持续经历，两者不是二选一。
- 120天内未完成100小时，通常需要在模拟机完成熟练检查并重新建立120天100小时。

### 12个月航线检查和6个月熟练检查

- 机长需要在前12个日历月内完成规定航线检查，用于证明能够履行机长职责。
- 担任飞行机组必需成员的驾驶员需要在前6个日历月内完成所服务机型的熟练检查。
- 手册可能允许在基准月前后一个日历月完成并视为基准月完成；这不等于有效期任意顺延。
- 检查不合格或超期后的恢复，应查技术管理手册和训练大纲第14章，不能直接补录日期。

### 机组搭配和100小时经历

- 安排机组时至少一名驾驶员应在该型别具有规定的航线飞行经历。该要求用于防止两名低经历人员形成弱搭配。
- A1/A2/B/C/D/E副驾驶和不同类别机长的搭配权限不同；巡航机长只能在巡航阶段替代相应职责。
- A1/A2能否操纵起降还取决于机场类别和机长是否具备相应教员资格。
- 不要只看到“某人超过100小时”就判定搭配合格，还要核对技术等级、教员资格、机场/航线、英语和近期经历。

### PF、PM和左右座

- PF表示主要操纵飞机，PM表示监控和协助；座位位置不能自动推出PF/PM权限。
- 副驾驶通常在右座履职，机长通常在左座履职，但训练、检查、巡航替代和教员带飞存在专门规则。
- A1与L1/L2航线教员搭组时的PF权限不同，必须查技术管理手册附录K。
- Z类巡航机长可在巡航阶段履职，但只有在满足手册规定的教员搭配等条件时才能操纵起降。

### 月100小时、年900小时不是全部限制

- 飞行机组成员在任一日历月和任一日历年都有累计飞行时间上限，当前样本用于定位的数字是月100小时、年900小时。
- 还要同时核对连续7日飞行值勤期、月飞行值勤期、单次飞行值勤期和连续值勤天数。
- 某人月飞行时间未超100小时，不代表其连续7日值勤、月值勤或当次任务一定合规。

### 10小时休息、144小时内48小时休息

- 前一飞行值勤结束到下一飞行值勤开始前，通常要求至少连续10小时休息。
- 执行飞行任务或主备份前的144小时内，需要安排规定的连续48小时休息。
- 跨越6个或以上时区返回基地时，还有单独的恢复要求。
- “连续48小时”还要结合是否包含正常昼夜节律下的完整夜间休息机会，不能只按两个时间戳相减。

### 扩编机组13小时和17小时

- 当前样本中，3名驾驶员和4名驾驶员扩编机组分别对应不同总飞行时间上限。
- 使用扩编上限前，必须核对合格的巡航替代人员、机上休息设施、休息安排和着陆前连续休息。
- 只增加一名人员但不满足休息设施和资格条件，不能直接套用扩编上限。

### 值勤延长和报告

- 意外情况可能允许机长和公司延长飞行值勤期，但普通延长、起飞后的意外情况和把飞机安全落到下一目的地/备降场的情形边界不同。
- 延长超过规定分钟数可能触发次数限制、局方报告和修正措施。
- 需要记录原计划、实际值、延长原因、机长决定、休息安排、报告日期和修正措施，不能只记录“延长2小时”。

### 特殊机场和专项运行

- 特殊机场资格通常同时要求机型经历、理论/模拟机熟悉、前12个月运行或航线检查，并存在12个月有效期。
- EDTO、RNP、RVSM、LVO、高原等不能只看人员训练记录，还要核对公司运行批准、飞机设备、机场天气和放行程序。
- LVO中的RVR数字属于具体起飞、滑行或进近场景；必须连同设备故障、机长/副驾驶经历和签注有效期读取。

### 体检中断30天和90天

- 因伤病连续中断飞行达到规定天数后，返回运行前需要重新进行相应体检鉴定。
- 30天以上不足90天与90天以上的检查范围不同，不能只凭仍持有纸面有效体检证安排飞行。
- 同时核对航医报告、医学资料、体检结论和重新取得合格证的状态。

3. 使用 `compare-manuals` Skill 建立全文顺序锚点和缺口，再按章人工复核。不得仅按修订记录抽样。
4. 建立逐章差异台账：新增、删除、内容修改、章节移动、表格变化、交叉引用变化和待确认。
5. 优先审查责任主体、批准权、机长权力、人员资格、数字、单位、天气标准、否定词、例外、报告时限和处理后果。
6. 第3、4、5、6、12、13、14章必须进行第二轮语义复核；第4章所有值勤和搭配表格逐格核对。
7. 同时核对运行手册变化是否要求更新技术管理手册或训练大纲，不能把三本书的不同表述直接判为冲突。
8. 对同一主题的多处条款建立引用链，确认总则、具体程序和例外没有漏读。
9. 输出两版完整原文和位置证据；无法判断是否替代或移动时标记待人工确认。
10. 新版确认生效后才更新本 Skill，删除失效的旧事实，不保留历史兼容口径。

## 输出要求

- 开头说明手册版本、读取范围、表格附件是否完整和未验证内容。
- 用“适用场景—责任主体—前提—规则—例外—报告/记录”结构回答。
- 具体数字保留原表头、单位、起算口径和适用机组，不脱表引用。
- 涉及人员资格或训练时，明确指出还需核对哪一本手册。
- 涉及实际航班时，区分计划值、实际值、延长原因和批准/报告记录。

## 禁止事项

- 不上传或提交原始运行手册。
- 不只读关键词附近一段就解释跨章节运行规则。
- 不把训练合格、执照有效、技术等级和运行授权混为一体。
- 不根据旧版记忆、行业常识或其他公司规则替代现行公司手册。
- 不为具体部门、机型、章节标题、机场或示例数字编写程序特判。
