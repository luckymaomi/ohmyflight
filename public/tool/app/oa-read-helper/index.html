自动点OA助手

自动打开 OA 待办页并自动点击“已阅”；自动点击百分之95以上oa，部分类型（如督办）需人工处理，自动点击失败时手动完成后等待刷新即可。

工具介绍

这是一个 Python + Playwright 脚本，用于自动处理 OA 待办列表。

自动打开 OA 待办页并自动点击“已阅”；自动点击百分之95以上oa，部分类型（如督办）需人工处理，自动点击失败时手动完成后等待刷新即可。

下载
下载 app.py
使用说明

安装依赖：

pip install playwright colorama
playwright install chromium

运行：

python app.py
