文件批量处理工具

批量重命名、创建文件夹、修改扩展名

工具介绍

这是一个基于 Python Tkinter 的图形化工具，用于批量处理文件。支持直接操作本地文件系统，速度快，适合处理大量文件。

主要功能包括：添加前缀、添加后缀、删除字符、替换字符、修改扩展名、创建同名文件夹、递归扫描子文件夹、预览结果、撤销操作。

处理10000个文件仅需5-10秒，内存占用低，支持百万级文件。

使用说明

环境要求：Python 3.6+（自带 tkinter，无需额外安装）

运行方式：

python file-batch-tool.py

操作步骤：

1. 点击"浏览"选择要处理的文件夹
2. 点击"扫描"加载文件列表
3. 选择操作类型并输入参数
4. 点击"预览"查看重命名效果
5. 确认无误后点击"执行操作"
6. 如需撤销，点击"撤销上次操作"

注意事项：操作前建议先备份重要文件。如果操作选项的输入框为空，则不会修改文件名。撤销功能只能撤销最近一次操作。修改扩展名只是重命名，不会转换文件格式。

使用示例

批量添加日期前缀：选择"添加前缀"，输入"2024_"，结果：photo.jpg → 2024_photo.jpg

创建同名文件夹：选择任意操作（可不填内容），勾选"创建同名文件夹并移动文件"，结果：report.pdf → report/report.pdf

批量修改扩展名：选择"修改扩展名"，输入".png → .jpg"，结果：image.png → image.jpg（注意：这只是重命名，不会转换图片格式）

删除特殊字符：选择"删除字符"，输入"_"，结果：my_file_name.txt → myfilename.txt
