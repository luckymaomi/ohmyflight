通用锁班助手

自动化填写飞行门户非生产任务录入表单

工具介绍

这是一个 Python 脚本工具，用于自动化填写飞行门户的非生产任务录入表单，支持批量录入锁班信息。

支持批量/手动模式、白名单过滤、可选统一备注、冲突检测等功能。

当前版本说明：
  • app.py - 原始助手：稳定单页流程，支持批量、手动、Excel 导入三种模式，实时写入结果 Excel
  • superapp.py - 并发助手：实验性并发录入，默认 20 个页面并发处理，适合大量数据提速
  • 启动时可选填写统一备注，输入 OK 确认后复用于本次全部记录
  • 并发助手会新建结果 Excel，每处理一条实时写入一条结果
  • 遇到冲突或错误时会自动跳过；冲突原因会写入结果 Excel 的备注和冲突列
