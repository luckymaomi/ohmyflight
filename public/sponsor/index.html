事情太多，无暇顾及？发给猫咪。

一句指令，自动分析，自动完成。

猫咪官网：https://luckymaomi.github.io/kitty/

安装猫咪：

npm install -g @jun133/kitty@latest

进入希望猫咪工作的项目目录，初始化配置：

kitty init

打开 .kitty/.env，填写 provider API key，然后启动：

kitty

不同日期的大量锁班工作，吃个饭回来，全部完成。

锁班皇帝读取 Excel 或批量文本，按人员、类型和日期逐条录入。成功、冲突和错误实时保存，中断后仍能继续核对。

人员越来越多，手动查起落数查到崩溃？睡个午觉回来，名单已经跑完。

起落数查询助手按员工号和日期区间，逐人查询飞行经历或左座经历与起落数，并把结果汇总到本地 Excel。

OA 99+ 无暇处理？让程序接手重复点击。

自动点 OA 助手处理所有可直接确认的“已阅”待办。需要意见、审批、签署或特殊处理的事项仍然交给人判断。

培训总表，不再靠人脑记住每个到期日。

培训皇帝把有效期、项目记录、培训规则和取消备注放进同一条判断链，集中找出已过期、必须安排、建议安排和已排未覆盖。

插入一段内容，为什么不会让后文全部飘红？

校对之王先用两侧唯一且完全一致的长文本建立锚点，再通过最长递增子序列保留顺序一致的锚点。锚点之间使用滑动窗口、2-gram 相似度和最大权重链完成对齐，最终识别参考新增、参考删除、内容修改和待确认。
